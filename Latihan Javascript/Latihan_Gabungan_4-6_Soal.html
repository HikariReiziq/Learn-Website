Tugas: Konverter Nilai Siswa

Buka konsol developer (Tekan F12 lalu klik tab "Console") untuk melihat hasil program Anda.

Soal

Buatlah program JavaScript untuk mengonversi nilai angka siswa (0-100) menjadi predikat huruf.

Tugas Utama

  1. Buat fungsi bernama konversiNilai yang menerima satu parameter: nilaiAngka.
  2. Gunakan if / else if / else untuk menentukan predikat sesuai ketentuan.
  3. Fungsi mengembalikan (return) teks predikat.

Aturan Predikat

  • Nilai >= 90 → A (Sangat Baik)
  • Nilai >= 80 → B (Baik)
  • Nilai >= 70 → C (Cukup)
  • Nilai >= 60 → D (Perlu Perbaikan)
  • Nilai < 60 → E (Gagal)

Data Uji

  • Budi: 85
  • Ani: 92
  • Candra: 58
  • Dewi: 75

Contoh Output

    --- Laporan Hasil Konversi Nilai ---

    Siswa "Budi" (Nilai: 85) mendapatkan predikat: B (Baik)
    Siswa "Ani" (Nilai: 92) mendapatkan predikat: A (Sangat Baik)
    Siswa "Candra" (Nilai: 58) mendapatkan predikat: E (Gagal)
    Siswa "Dewi" (Nilai: 75) mendapatkan predikat: C (Cukup)

    --- Proses Selesai ---
